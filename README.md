- Simple and clean UI
- Win detection and reset option
- Built with Java and XML

## Tech Stack

- **Language:** Java  
- **UI Design:** XML  
- **IDE:** Android Studio  
- **Build Tool:** Gradle  

## How to Run

1. Clone or download this repository
2. Open it in Android Studio
3. Let Gradle sync automatically
4. Run the app on an emulator or physical device
